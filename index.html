Using K-means and PCA
for Geochemical Exploration Targeting

Southeast Ireland is known for its diverse geology and associated mineral deposits. By utilising K-means, we can cluster the stream sediment sample data to reflect the lithology underneath, and find out where the LCT Pegmatite deposit. To further validate the potential location of anomaly of LCT Pegmatite deposits, Principal Component Analysis (PCA) can be applied using specific enriched elements in these deposits. The data used in this project is made available by Geological Survey Ireland.
